비정형 문서를 XML 형태로 변환하는

Doc Structurer

  • 메타정보 학습을 통해 비정형 문서를 정형 데이터로 변환하는 솔루션입니다.
  • 다양한 문서 형태를 이해하고 이를 XML 형식으로 변환하여 데이터의 통합과 활용을 용이하게 합니다.

라벨링 문서기반 데이터를 추출하는

TS Expert

  • 워드, 한글 문서 등 비정형 데이터에서 정형데이터로 변환하여 제공합니다.
  • 지정된 특정 키워드에 해당하는 데이터 추출로 효율적인 업무처리를 지원합니다.

두 개의 문서를 비교, 대사하는

Doc CompareX

  • 두 개의 문서를 비교하여 변경된 내용을 확인하고 관리하는 솔루션입니다. 변경된 부분만 파일로 추출하여 관리하고, 수정 각주와 메모 삽입 기능을 통해 피드백을 작성할 수 있습니다.

신규 문서를 자동 생성하는

Doc Generator

  • 새로운 문서를 자동 생성하는 솔루션으로 사용자가 입력한 조건에 따라 복잡한 조건의 문서도 신속하게 표준 문서로 생성할 수 있습니다. 엑셀 템플릿을 사용하여 간편하게 문서 생성, 조건 변경 및 오류 수정이 가능합니다.
